getInventoryAvailableTextFieldKeys
The method returns a list of product text fields that can be overwritten for specific integration.


Input parameters
inventory_id	int	Catalog ID. The list of identifiers can be retrieved by the getInventories method (inventory_id field).



Output data
The method returns the data in JSON format.
status	varchar(30)	SUCCESS - request executed correctly
ERROR - an error occurred during an API request. Error details will be described in 2 additional returned fields: error_message and error_code
text_field_keys	array	A list containing product text fields, where the key is the code of the text field and the value is the text field name
