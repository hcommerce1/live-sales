setInventoryDocumentStatusConfirmed
The method allows you to confirm an inventory document, which will affect the stock levels in the warehouse.


Input parameters
document_id	int	Document identifier



Output data
The method returns the data in JSON format.
status	varchar(30)	SUCCESS - request executed correctly
ERROR - an error occurred during an API request. Error details will be described in 2 additional returned fields: error_message and error_code



Sample
Input data:
{
        "document_id": 101
}
 Output data:
{ "status": "SUCCESS" }
